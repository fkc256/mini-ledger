# 🧾 나만의 미니 가계부 (Streamlit 기반 지출 관리 서비스)

Streamlit을 활용하여 만든 "간단한 개인 가계부 웹 애플리케이션" 입니다.  
지출·수입을 등록하고, 필터링 및 카테고리별 지출 통계를 시각화할 수 있습니다.

## 📌 주요 기능

### 1. 거래 등록
- 날짜 선택 (캘린더 기반 입력)
- 구분 선택: 지출 / 수입
- 카테고리, 내용 텍스트 입력
- 금액 입력
- 입력값 검증 (카테고리·내용 필수)

### 2. 거래 목록 조회
- 전체 거래 내역 테이블 표시
- 날짜, 구분, 카테고리, 내용, 금액 확인 가능

### 3. 필터 기능
- 기간 필터 (시작일 ~ 종료일)
- 내용 키워드 검색
- 구분 필터 (전체 / 지출 / 수입)
- 카테고리 필터

### 4. 요약 통계
- 총 수입
- 총 지출
- 현재 잔액 자동 계산

### 5. 카테고리별 지출 통계
- 지출 데이터만 집계
- 카테고리별 지출 합계 Bar Chart 시각화
- 지출 합계 테이블 제공
